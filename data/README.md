# Data

Raw data is not included in this repository due to Bloomberg licence restrictions.

## Equity price data

Daily adjusted closing prices for six large-cap US technology stocks sourced from Bloomberg Terminal.

| Ticker | Company | Sample |
|---|---|---|
| AAPL | Apple Inc. | Jan 2014 – Dec 2025 |
| MSFT | Microsoft Corp. | Jan 2014 – Dec 2025 |
| IBM | IBM Corp. | Jan 2014 – Dec 2025 |
| NVDA | NVIDIA Corp. | Jan 2014 – Dec 2025 |
| GOOGL | Alphabet Inc. | Jan 2014 – Dec 2025 |
| AMZN | Amazon.com Inc. | Jan 2014 – Dec 2025 |

2,997 daily log-return observations. Portfolio returns are equally weighted.
